Mondrian Grid – это профессиональное приложение для анализа цвета
и работы с палитрами, разработанное на Python. Программа предоставляет
комплексный набор инструментов для извлечения доминирующих цветов из
изображений, генерации цветовых гармоний и создания тематических схем.
Интерфейс выполнен в тёмной цветовой схеме с акцентами индиго, что
обеспечивает комфортную работу при длительном использовании.

Приложение поддерживает анализ изображений с помощью алгоритма K-means
для точного определения основных цветов, классификацию настроения палитры
и генерацию гармоний по различным схемам (аналогичная, комплементарная,
триадная и другие). Mondrian Grid оснащено продвинутыми инструментами
для конвертации между цветовыми моделями (HEX, RGB, HSL) и проверки
контраста по стандартам WCAG.

Mondrian Grid отличается оптимизированной архитектурой – оно включает
систему истории палитр с автоматическим сохранением, конструктор тем
для веб-дизайна и поддержку экспорта в различные форматы (CSS, SCSS,
GIMP Palette, JSON, PNG). Приложение подходит как для быстрого анализа
изображений, так и для профессиональной работы с цветом в дизайне
и разработке интерфейсов.
